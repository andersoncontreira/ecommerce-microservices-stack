# Testes de Carga

## Instalação das dependencies
 - Node 16+
 - Artillery

### Configurando o node
Para configurar o node, use os comandos abaixo:
```bash
  nvm install 20
  nvm use 20
```

### Instalando o Artillery
Para instalar as dependencias do projeto execute o comando a seguir:
```bash
  nvm use 20
```

```bash
  npm install
```
